동기 통신 vs 비동기 통신

동기통신은 프로그램의 흐름에 따라서 통신이 완료가 될때까지 나머지 프로그램이 대기
let number = ( 어떤 서버와 통신을해서 가져오는 값 );
number 라는 값이 준비될때까지 대기
그 밑에 줄부터는 실행을 안한다.

let number = ( 서버로부터 값을 받아오는 코드 ); // 기다린다.
// 서버에서 만약 6이라는 값을 돌려줬다면
if (number == 5) {
    ...
}

비동기통신은 통신은 그대로 실행하고, 프로그램의 나머지 코드는 그대로 실행
통신 => 서버에 요청을 보낸다.
client  =====>   server
(우리)           (서비스 공급자(naver, google))

동기는 server 에서 보낸 응답을 기다렸다가 실행하는것
비동기는 요청을 보내고 응답 여부와 상관없이 나머지도 실행

비동기에서도 server가 보내준 값을 대기하는것은 같다.
let number  = (서버에서 받아오는 코드); 
// number를 서버에서 받아오기로 약속을 한다.
// number는 서버에서 나중에 받아오기로 약속을 해놓았군.
// number를 사용하는 코드를 제외하고 나머지 코드를 먼저 실행
if (number == 5) {
 ...
}
